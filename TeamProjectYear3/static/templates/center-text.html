Lifelog Memories Server


The Lifelog Memories Server (LMS) is a module of Glasgow Memories Server. LMS comprises of lifelogs (i.e. data) generated by wearable sensors, which are commonly known as lifelogging devices such as wearable cameras (e.g. Autographer- passively capturing images) and GPS Devices (tracking location of the users). In this page, you will be viewing a small collection of:
  • Lifelog images captured by Autographer
  • Videos captured using Google Glass
  • Heat map, which is generated using the GPS data obtained from a number of lifeloggers
Please Register and Login using the Log In option in the topmost panel to take advantage of all the functionalities offered by our application
